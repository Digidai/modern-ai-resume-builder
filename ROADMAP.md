# Roadmap

This roadmap outlines current priorities for ModernCV. Items are grouped by
expected delivery horizon and may change based on community feedback.

## Now

- Expand role examples to cover more job families.
- Add JSON import to complement JSON export.
- Improve template previews and add print QA checklist.
- Add basic CI for build and data validation.

## Next

- ATS guidance and resume scoring hints.
- Cover letter generator and multi-document exports.
- Localization and multi-language templates.
- Resume version history with restore points.

## Later

- Team and HR features (shared template kits, review workflows).
- Integrations with ATS or HRIS platforms.
- Collaboration and comment threads.
- Analytics for template usage and conversion.
